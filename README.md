# 📱 WDROID

> **wdroid** - An operating system written in JavaScript, CSS, and HTML, attempting (REALLY ATTEMPT) to replace Android.

---

## 🌐 About WebSoft
**WDROID** is the flagship project of **WebSoft**. It is a high-performance, minimalist Web Operating System (WOS) designed to run entirely locally. Our mission is to provide a clean, fast, and functional environment that prioritizes user utility and execution speed.

## 🚀 Key Features
*   **Pure Web Tech:** Built strictly with HTML5, CSS3, and Vanilla JavaScript.
*   **Fully Local:** Operates 100% on the client-side(Not really) for maximum privacy and instant response times.
*   **Independent Architecture:** A standalone system designed to function without external dependencies.

## 🛠️ Tech Stack
*   **Logic:** JavaScript (Vanilla)
*   **Styling:** CSS3
*   **Structure:** HTML5
